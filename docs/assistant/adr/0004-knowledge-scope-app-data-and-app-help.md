# Knowledge scope limited to the user's data + app help; general/legal refused

The assistant answers only two classes of question: facts about the user's own
authorized app data, and how to use the app itself. Anything else — general knowledge,
and especially legal/rights questions ("what are my rights if the syndic ignores my
complaint?") — is refused with a fixed deflection line that points the user to file a
complaint. We chose this narrow scope because a property-management assistant that
dispenses legal advice would take on real liability for answers it might hallucinate,
across three languages and three jurisdictions (Morocco, Algeria, France) with different
property law. Refusal is cheaper than being wrong here.
